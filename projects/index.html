Lung Nodule Detection using U-Net++ and 3D-CNN

Timeline: July 2024 – Present

Advisor: Prof. Peng Zhang

This project focuses on enhancing the accuracy of lung nodule detection from CT scans using deep learning techniques. Initially, the U-Net architecture was used for segmentation, but it suffered from high false-positive rates. To address this, I incorporated a 3D Convolutional Neural Network (3D-CNN) for post-processing, significantly improving detection precision.

Building upon this baseline, I integrated a more advanced U-Net++ structure with 3D-CNN, resulting in superior segmentation and classification performance. The model was trained and evaluated on publicly available lung CT datasets, with performance metrics including Dice coefficient and precision-recall scores.

Tools & Technologies: Python, Linux, PyTorch, NumPy, 3D Slicer

Facial Recognition via Template Matching

Timeline: Oct 2024 – Dec 2024

Course Project: Digital Image Processing

This project was aimed at implementing a facial recognition system using classical image processing techniques. I started with preprocessing steps including grayscale conversion, image normalization, and resizing (252×256 pixels).

I applied template matching to detect eye regions using predefined left and right eye templates, followed by Hough Transform to accurately locate pupils. Based on the pupil positions, facial alignment and rotation were performed, and the face region was cropped to a standardized format suitable for recognition tasks.

Tools & Technologies: MATLAB
